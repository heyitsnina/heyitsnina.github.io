How to Format Your Content: The Screening Room

Powerpoints

Videos and Links in Your Powerpoint

All video files and links should be embedded within a Microsoft PowerPoint (.pptx) presentation slide and set to play automatically when the slide is selected (Video Tools > Playback > Start > Automatically).

Tutorial »

Images Formats

All images should be either JPG or PNG.

Tutorial »

Your Presentation Aspect Ratio

ALL CONTENT needs to have an ASPECT RATIO OF 16:9 to be displayed correctly on our presentation screens! See examples below.

  • Microsoft Powerpoint - DESIGN TAB > SLIDE SIZE > WIDESCREEN 16:9
  • Video Resolutions - 1280 X 720, 1920 X 1080
  • Image Resolutions - 1280 X 720, 1920 X 1080, 3840 X 2160

File Names

Single Presentations

Please label presentations in the following format:

  • 01_Presenter Name or Company

Multiple Presentations

Combine ALL of the presentations into ONE Master Deck. If you are unable to create a Master Deck, the AV tech will create the master deck however, are not liable for ANY resulting formatting issues.

Please label Master Decks in the following format:

  • 00_MASTER_Event Name

Screening Room Content

Content that will be displayed on the Hiperwall should be labelled according to which screens they go on. Use the provided content map to accurately label the positions of each piece of content.

Please label presentations in the following format:

  • S1_File Name or (S1+2_File Name) for same image on multiple displays.
  • S2_File Name
  • S3_File Name
  • S4_File Name
  • MAIN_File Name
